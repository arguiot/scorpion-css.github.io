Simpler, more semantic code

We believe that the best developers are the lazy developers. That's why we encourage simplicity in our components, so that you can write less, to achieve the same results. To make your components even simpler to write, we highly encourage the usage of the jade templating language.

Fully responsive

Scorpion components scale flawlessly, so that they can be used on any size device, from a phone to a desktop !

Sassy as can be

Scorpion is a css (and js) library, written entirely in Sass. You can quickly get started with our precompiled css, or build upon the sass source code.

Readable HTML

We made our framework as semantic as we could, to make the code you write more readable, and more pleasant to write.
